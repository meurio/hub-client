# Understanding the Client Setup

Build and tests based on [Create React App with Typescript](https://create-react-app.dev/)

**Commands**

- Tests:

```sh
pnpm m run tests # pnpm m run coverage
```

- Development server:

```sh
pnpm m run start
```

- Production build:

```sh
pnpm m run build
# pnpm m run start
```
